# Word Mix Bot

A Telegram bot that provides mixed-up words for users to unscramble.

## Features
- Sends mixed-up words
- User interaction
- Score tracking

## Technologies Used
- Python 3
- Telegram Bot API

## Installation

```bash
git clone https://github.com/havvakrbck1/word-mix-bot.git
cd word-mix-bot
pip install -r requirements.txt
python bot.py
